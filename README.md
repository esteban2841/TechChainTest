# Tech chain landing page (Vite + React + TS + Tailwind)

## Scripts
- `pnpm dev` / `npm run dev` — Dev server
- `pnpm build` / `npm run build` — Build
- `pnpm preview` / `npm run preview` — Preview production build

## Stack
React 18, TypeScript, TailwindCSS, Framer Motion, Vite.

## Structure
See `src/components` for Atomic Design: atoms, molecules, organisms; and `src/templates/LandingPage.tsx`.

## Getting Started
```bash
pnpm i   # o npm install / yarn
pnpm dev # o npm run dev
```
